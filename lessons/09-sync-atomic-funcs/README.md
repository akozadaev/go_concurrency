# Атомик-счётчик (`sync/atomic`)

## Идея

Несколько горутин увеличивают общий счётчик **`int64`** через **`atomic.AddInt64`**; в конце значение читается **`atomic.LoadInt64`**. Операции атомарны относительно друг друга - гонки по полю `counter` нет.

## Запуск

```bash
cd lessons/09-sync-atomic-funcs
go run .
go run -race .
```

`-race` не должен находить гонок по счётчику (в отличие от обычного `counter++` без синхронизации).

## Замечание

Сравните с уроком **`08-sync-atomic-int64`** (`atomic.Int64`). Здесь - классические **`AddInt64` / `LoadInt64`**.

# Горутины без синхронизации

Две горутины печатают цифры и буквы с задержками, **`main` сразу выходит**. Вывод может **не появиться** - типичная демонстрация: дочерние горутины не "привязаны" к жизни процесса.

## Запуск

```bash
cd lessons/01-goroutines
go run .
```

Попробуйте раскомментировать `time.Sleep` в коде и сравнить поведение.
